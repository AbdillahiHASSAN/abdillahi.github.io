Color in HTML and CSS

Colors are a fundamental part of web design, used to enhance visual appeal and convey meaning.

In HTML, colors can be applied using inline styles or through attributes like bgcolor.

In CSS, colors are styled using the color property (for text) and background-color property (for backgrounds).

Ways to Specify Colors in CSS

Type Example Description
Named Colors red, blue, green Predefined color names available in CSS.
Hexadecimal Codes #FF5733 A six-digit code representing red, green, and blue values.
RGB and RGBA rgb(255, 87, 51) / rgba(255, 87, 51, 0.5) Specifies red, green, and blue values, with optional alpha transparency.
HSL and HSLA hsl(12, 100%, 60%) / hsla(12, 100%, 60%, 0.5) Defines color using hue, saturation, lightness, and optional alpha transparency.

Colors bring life to your web pages, but they should be chosen carefully to ensure accessibility and good contrast!

Typography in HTML and CSS

Typography refers to the art of arranging text to make it readable and visually appealing. It focuses on styling elements like font choice, size, spacing, and alignment, which play a key role in effective web design

property Description Examples values
Font Family Determines the style of the text. Serif, Sans-serif
Font Size Specifies how large or small the text appears. Small, Medium, Large
Font Weight Controls the thickness or boldness of the text. Light, Regular, Bold
Line Height Adjusts the spacing between lines of text. Tight, Normal, Relaxed
Text Alignment Aligns the text within its container. Left, Center, Justified

Typography is crucial in web design as it helps create a hierarchy of information, improves readability, and enhances the overall look of a webpage.

Tables in HTML and CSS

Tables are used to display data in a structured format, organized into rows and columns. They are essential for presenting data like pricing, schedules, or comparisons in a clear and concise manner.

Element Description Example Use
Table The overall structure that contains all data. Organizes content systematically.
Row A horizontal group of data cells. Represents a single line of data.
Header Highlights key labels for each column. Used for titles or descriptions.
Data Cell Displays individual pieces of data. Contains text, numbers, or symbols.
Caption Provides a title or summary for the table. Explains the tableâ€™s content.

Tables are a powerful tool for structuring information but should be used only when presenting tabular data. For layout purposes, modern CSS techniques are preferred for flexibility and responsiveness.
